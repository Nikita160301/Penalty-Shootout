# Penalty Shootout Game

## Description
This is a simple web-based penalty shootout game where two players take turns trying to score goals against a computer-controlled goalkeeper. The game allows players to enter their names and keeps track of the scores. The game ends when one player reaches 10 goals, and the winner is displayed.

## How to Play
1. Open the `index.html` file in a web browser to start the game.

2. When the game loads, a popup will appear asking players to enter their names. Enter Player 1's name and click "Next."

3. Enter Player 2's name and click "Start Game."

4. The game will start, and players can take turns trying to score goals by clicking the various Buttons. The goalkeeper will move to different positions to block the shots.

5. The game keeps track of the scores, and when one player reaches 10 goals, the game ends, and the winner is displayed.

6. To start a new game, you can refresh the page.

## Files
- `index.html`: The main HTML file that contains the game interface.
- `style.css`: The CSS file that styles the game interface.
- `script.js`: The JavaScript file that contains the game logic and functionality.

## Features
- Two players can enter their names to personalize the game.
- Players can take turns to shoot penalty kicks.
- A computer-controlled goalkeeper attempts to block the shots.
- The game keeps track of each player's score.
- The game ends when one player reaches 10 goals, and the winner is displayed.
- A pop-up window is used for player name entry and game end notification.

- You can customize the game by changing the images, animations, and styling in the CSS file.
